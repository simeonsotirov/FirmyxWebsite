# FirmyxWebsite

Firmyx landing page — AI-powered financial risk analysis for SMBs.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- [Next.js 16](https://nextjs.org) (App Router)
- TypeScript
- Tailwind CSS
- `next-themes` for dark/light mode
- Lucide React for icons

## Fonts

This project uses the Inter font via Google Fonts (loaded in `globals.css`).

## Deploy

Deployable to [Vercel](https://vercel.com) or [Netlify](https://netlify.com) with zero configuration.
